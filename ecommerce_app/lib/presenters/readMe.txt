phần này dùng để:
    - Các file ở mục này đăt tên theo quy tắc: <tên file model tương ứng>_presenter.dart
    - Tên class: <model class name>+Presenter
    - Tạo các class tương ứng với các class bên model, ví dụ: ProductPresenter,...
    - Dùng để lấy dữ liệu từ các class bên model rồi truyền cho view, ví dụ lấy product list
